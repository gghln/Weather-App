# Weather App

A simple React weather application that fetches weather data from the OpenWeather API based on user input.

## Setup

1. Get an API key from [OpenWeatherMap](https://openweathermap.org/api).
2. Create a `.env` file in the root directory and add your API key:
   ```
   VITE_OPENWEATHER_API_KEY=your_actual_api_key_here
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm run dev
   ```

## Features

- Search for weather by city name
- Displays current temperature, weather description, and icon
- Shows feels-like temperature and humidity
- Responsive design with Tailwind CSS

## Technologies Used

- React
- Vite
- Tailwind CSS
- OpenWeather API
